Customise the HTML banner inserted into HTML-format emails. Leave blank to use the auto-generated banner (controlled by the settings above).
The banner element must include id="org-security-banner" so the extension can replace it on updates.
Available placeholders: {{cls}} classification • {{colour}} hex colour • {{bil}} BIL number (0–5) • {{backgroundCss}} solid colour or hatched gradient (per classification setting) • {{textColour}} #000 or #fff for contrast against classification colour • {{caveats}} formatted caveats • {{accessLine}} access line with leading <br> (empty if none) • {{expiresLine}} expires line with leading <br> (empty if none)
